# 新电脑快速配置指南

## 🚀 一键安装（最简单）

### Windows 用户

**方法 1: 双击运行（推荐）**
```
双击运行: 一键安装.bat
```

**方法 2: PowerShell**
```powershell
.\一键安装.ps1
```

### 就这么简单！

安装脚本会自动：
1. ✅ 检查 Python
2. ✅ 安装依赖库（requests, supabase）
3. ✅ 复制文件到系统目录
4. ✅ 配置 PATH 环境变量

## 📋 安装后

### 1. 重启终端

关闭并重新打开 PowerShell 或 CMD

### 2. 测试命令

```powershell
cloud-config
```

如果看到配置导出成功，说明安装完成！

## 💡 使用示例

```powershell
# 导出所有配置为 config.json
cloud-config

# 导出到指定文件
cloud-config --output my_config.json

# 只导出 PATH 配置
cloud-config --group path_config
```

## 🐛 如果遇到问题

### 问题：命令无法识别

**解决：**
1. 确保已重启终端
2. 或使用完整路径：`%USERPROFILE%\tools\supabase-tools\cloud-config.bat`

### 问题：缺少依赖库

**解决：**
```powershell
pip install requests supabase
```

### 问题：Python 未安装

**解决：**
1. 下载安装 Python: https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
